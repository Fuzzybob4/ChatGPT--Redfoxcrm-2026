# RedFox CRM 2026 MVP

A focused MVP for RedFox CRM: dashboard, customers with multiple service properties, jobs/scheduling, invoice add-ons, customer portal flow, and iOS SwiftUI starter.

## Stack

- Next.js / React web MVP
- Supabase-ready data layer
- Stripe-ready payments
- Resend-ready invoice email hooks
- SwiftUI iOS starter screens

## Required environment variables

```bash
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
STRIPE_SECRET_KEY=
RESEND_API_KEY=
```

## MVP focus

1. Dashboard
2. Customers + multiple properties/service areas
3. Jobs / Schedule
4. Invoices + customer-selectable add-ons
5. Settings / integrations
